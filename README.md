# MathCalculator

# Умный Консольный Калькулятор 

Моя контрольная работа по дисциплине **«Объектно-ориентированные системы программирования»**. 
Калькулятор умеет решать сложные примеры со скобками, запоминать историю операций и сохранять результаты в файл.

## 📝 Описание архитектуры
Программа написана на языке Java с использованием паттерна **MVC** (Модель-Вид-Контроллер), чтобы код был аккуратным и разделенным на логические части:
* `MathEngine.java` (Модель) — математическое ядро. Здесь происходят вычисления и работа с текстовыми файлами истории.
* `ConsoleScreen.java` (Представление) — отвечает за то, что пользователь видит в консоли (меню, сообщения).
* `SystemController.java` (Контроллер) — обрабатывает выбор в меню и связывает интерфейс с ядром.
* `RunApp.java` — стартовая точка программы.

Также в проекте используется **Maven**, чтобы автоматически подключить классную библиотеку `exp4j` для решения примеров с любым количеством скобок и знаков.

## Как запустить проект (Важно для проверки!)

Так как я использую Maven для подгрузки библиотеки `exp4j`, обычный запуск файла может выдать ошибку. Пожалуйста, следуйте этой инструкции:

1. Откройте скачанный проект в вашей среде разработки (например, IntelliJ IDEA).
2. Обязательно дайте IDE загрузить зависимости! В IntelliJ IDEA появится маленькая иконка с буквой **"m"** (Load Maven Changes) — нажмите на неё. Либо во вкладке Maven справа нажмите **Reload All Maven Projects**.
3. Когда все пакеты загрузятся из файла `pom.xml`, найдите файл `RunApp.java`.
4. Запустите метод `main` в файле `RunApp.java`.

## Что умеет программа?
1. **Решать примеры:** поддерживает сложение, вычитание, умножение, деление и скобки. Например: `(2 + 2) * 2`.
2. **Вести историю:** все успешные решения сохраняются в локальный файл `my_calc_memory.txt` и не пропадают при перезапуске.
3. **Сохранять логи:** можно выгрузить всю историю в отдельный текстовый файл, указав его название.
